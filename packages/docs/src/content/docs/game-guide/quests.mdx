NetCrawl's quest system guides you through programming concepts while unlocking new game features. Quests are organized into 8 thematic **chapters**, each teaching different skills.

## Quest Structure

Each chapter has:
- **Mainline quests** -- sequential gates that must be completed in order
- **Side quests** -- optional branches that unlock special rewards
- **Prerequisites** -- quests that must be completed before a quest becomes available

Quests have **objectives** (things to accomplish) and **rewards** (what you get for completing them).

|-------|---------|-----------|------------|

## Reward Types

Quests provide several types of rewards:

| Reward Type | Description | Example |
|-------------|-------------|---------|
| **Resources** | Data, RP, or credits | 500 data |
| **Passive Effects** | Permanent global bonuses | +5% harvest speed |
| **Items** | Equipment for workers | 1x Iron Pickaxe |
| **Recipe Unlocks** | New crafting recipes | Scanner recipe |
| **Chips** | Installable upgrades | Defense chip (rare) |
| **Unique Equipment** | One-of-a-kind powerful items | Full Stack Pickaxe (5.0x) |

:::tip
Do not skip side quests! They often provide unique equipment or passive bonuses that are unobtainable any other way. The **Antivirus Module** and **Memory Allocator** are particularly valuable.
:::

## Programming Concepts Progression

The quest system is designed to teach real programming concepts in order:

1. **Ch. 1** -- Methods, attributes, conditionals, operators, loops
2. **Ch. 2** -- Event loops, state management, concurrency
3. **Ch. 3** -- Graph theory, pathfinding, topology
4. **Ch. 4** -- Conditionals (security), input validation, authentication
5. **Ch. 5** -- Infrastructure as code, microservices, API design
6. **Ch. 6** -- Performance profiling, caching, algorithmic complexity
7. **Ch. 7** -- High availability, load balancing, horizontal scaling
8. **Ch. 8** -- Software architecture, full-stack development

Each concept is introduced through gameplay mechanics that mirror the real-world application of that concept.
