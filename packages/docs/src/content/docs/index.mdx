## What is NetCrawl?

NetCrawl is a **programmable idle game** where you write real code to automate workers on a network graph. Your workers mine resources, solve puzzles, handle API requests, and defend against infections -- all driven by the code you write.

### How it works

1. **Write worker classes** using the Python or JavaScript SDK
2. **Register them** with the NetCrawl code server
3. **Deploy workers** from the game UI, assigning edges and equipment
4. **Watch them run** -- your `on_loop()` / `onLoop()` method executes repeatedly
5. **Progress** through 8 chapters of quests that teach real programming concepts

### Architecture

```
Your Code (Python/JS)       Game Server         Browser UI
 +------------------+      +-----------+      +----------+
 | WorkerClass      | ---> | Deploy    | ---> | Network  |
 | - on_startup()   |      | Queue     |      | Graph    |
 | - on_loop()      | <--- | Actions   | <--- | Controls |
 +------------------+      +-----------+      +----------+
       NetCrawl App            REST API          React App
```
